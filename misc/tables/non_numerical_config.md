When using **non-numerical color coding**, the following additional parameters must be defined in the [CATALOG_DATA] section:

| Parameter          | Possible values                                  | Description |
|--------------------|--------------------------------------------------|-------------|
| `groups`           | Any category of the selected **color_coding**    | Categories to be highlighted (e.g. *Kepler*, *K2*, *TESS*) |
| `color_groups`     | Any valid color or list of colors                | Colors assigned to each group |
| `plot_all_planets` | True or False                                    | Whether to plot planets that do not belong to any selected group (**default**: True) |
